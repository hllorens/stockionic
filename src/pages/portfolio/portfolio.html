Help Stockionic has two main functionalities:
  • Stock search and sort
  • Stock email alerts
Both are based on fundamental analysis features of stocks. Stockionic is intended avoid you the need of being constantly monitoring or analyzing stocks so you can just wait for email alerts to come to you.
Note: the list of stocks is limited. To add more contact hectorlm1983@gmail.com.

Fundamental values used
  • Value: current value of the stock
  • Value change %: current % change of the stock.
  • PER: Price to earnings (P/E) ratio. The lower the better
  • YLD: Anual dividend yield % and the number of dividends per year. The higher the better
  • YPR: Yeld divided by PER, this will be higher for stocks with higher dividend and lower per. I.e., the most profitable ones in terms of the earnings they have and the proportion they share with stock holders. The higher the better
  • 52 weeks value position: the current position in the 52 weeks value variation range. E.g., if min=$1 and max $1 and current value is $1.5 it is 50% in the range
  • 52 weeks volatility (52W VO): how much the value varied proportionally in 52 years. E.g., if you see 2x it means that the value doubled or shrinked to the half
  • EPSD: what % the EPS differed since last time it was reported. E.g., if in Q1 EPS was 0.5 and now it is 1 then EPSD=2x (the earnings doubled). There are some symbols for EPS evolution:
    • Positive evolution: / (two consecutive increases) v (incriease after a decrease)
    • Negative evolution: ^ (decrease after an increase) \ (two consecutive decreases)
The main mesure we use for sorting the stocks by default is YPR since it gives a good balance between dividend and earnings.
Note: the data comes from google.com/finance.

Stock search and sort
At the beginning you see a list of stocks. You can:
  • Search for stock symbols, tickers or names in the search bar ()
  • Sort the list using the buttons
    • YPR (used by default)
    • PER, 2nd click EPSD
    • YLD
    • 52W, 2nd click reverse order 52Wv, 3rd click volatility 52Wvo
    • Alerts only: show only stocks for which you have alerts (see next section for more information)


Stock email alerts ()
To add alerts you must click on the stock of your interest (e.g., GOOG). A screen with only that stock will be opened and you will see form input fields to add alerts. You can add alerts for the cases where a value is lower than (left) or higher than (right) what you want. The values for which you can add alerts are:
  • Value
  • Value Change %
  • PER
  • Yield
  • EPS
E.g., if you set an alert for GOOG Value low at $900 if it ever goes down that value the alert will fire.

When an alert fires you will receive an email and in the app you will see the stock highlighted in a red background. Furthermore, in the app, when you click on the alerts button () you will see the stocks for which an alert has fired first. To stop receiving emails or seeing the red background you can remove the value of the alerted field or change the value.

How to use the app?
Some use examples.
  • You want to invest in a stock but only when it reaches certain params: You would set some alerts and wait for emails
  • You don't know which stock to invest on (which ones could be better): You would do some sortings to pick them e.g., pick the first ones sorting by YPR which should be the most profitable ones in terms of fundamental analysis, or even look for positive EPS evolution like / or v
  • You monitor stocks to decide when to sell: Like in the first case you would set some alerts and wait for emails
