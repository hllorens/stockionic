Help Stockionic has two main functionalities:
  • Stock search and sort
  • Stock alerts
Both are based on fundamental analysis features of stocks.
Note: the list of stocks is limited. To add more contact hectorlm1983@gmail.com.

Fundamental values used
  • Value: current value of the stock
  • Value change %: current % change of the stock
  • PER: Price to earnings (P/E) ratio
  • Yield: Anual dividend yield % and the number of dividends per year
  • 52 weeks value (heat): the current position in the 52 weeks value variation. E.g., ...
  • 52 weeks volatility: how much the value varied proportionally in 52 years. E.g., if you see 2x it means that the value doubled or shrinked to the half
  • EPS diff: what % the EPS differed since last time it was reported


Stock search and sort
At the beginning you see a list of stocks. You can:
  • Search for stock symbols, tickers or names in the search bar
  • Sort the list using the buttons
    • YPR (used by default)
    • PER, 2nd click EPS diff
    • Yield
    • 52 weeks value, 2nd click reverse order, 3rd click volatility
    • Alerts only: show only stocks for which you have alerts (see next section for more information)


Stock alerts ()
To add alerts you must click on the stock of your interest (e.g., GOOG). A screen with only that stock will be opened and you will see some form input fields to add alerts. You can add alerts for the cases where a value is lower or higher than what you want. The values for which you can add alerts are:
  • Value
  • Value Change %
  • PER
  • Yield
  • EPS
When an alert fires you will receive an email and you will see the stock highlighted in red. When you click on the alerts button you will see the stocks for which an alert has fired first.

How to use the app?
Let's describe some use examples.
  • You want to invest in a stock but only when it reaches certain params
  • You don't know which stock to invest on. You want to see which ones could be better for you
  • You monitor stocks to decide when to sell
